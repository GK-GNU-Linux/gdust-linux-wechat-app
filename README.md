## 分支管理

```
gxgk-wechat-app
  ├─ master       // 默认分支（开发测试版本所用分支）
  ├─ stable       // 稳定分支（正式版本所用分支；高级保护分支，只允许管理员操作，通常由master分支-Merge而来）
  └─ [other]      // 其他开发分支（只允许该项目Collaborators创建及push分支）
```
